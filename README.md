# Weather_visualization
*気象データの可視化*

はじめに  
・使用したもの  
 -python3(jupyter notebook)  
 -openweathermap 無料APIライセンス  

内容  
-openweathermapから気象データを抽出し、日本の気温と風向、風力を地図上で可視化しました。
